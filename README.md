# wattsy-discord-bot

## Setup
Install python3 (depending on your OS).

In case of debian distrib
`
sudo apt-get install -y libffi-dev libnacl-dev python3-dev
`

After that you can create your venv (this is to make sure you are not mixing all your dependencies from differents project) \
`python3 -m venv wattsy-env`

Now you juste need to install your dependencies with \
`pip install -r requirements.txt`
